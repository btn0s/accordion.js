accordion.js
  • what is it?

    • accordion.js is a lightweight jQuery plugin for building multi-tiered, 'accordion style' dropdown menus.

  • how do I use it?

      just give your 'body' an id of 'closer' and wrap your menu in a 'menu' class, and build the menu with 'ul' lists and 'a' elements as the title bars

    • is that all I need?

        yep! styling is up to you, but accordion.js will handle the rest

  • where can I get it?

    • you can download the accordion.js file right here, or check out the github repo for a bit more info
