accordion.js
  • what is it?

    • accordion.js is a lightweight jQuery plugin for building multi-tiered, 'accordion style' dropdown menus.

  • how do I use it?

      just put a link to the file in your HTML, and build your tiered menu with 'ul' elements in plain HTML

    • is that all I need?

        other than that you'll need to put the class 'toggle' on the link elements that will have child lists, but that's it! accordion.js does the rest.

  • where can I get it?

    • you can download the accordion.js file right here, or check out the github repo for a bit more info
